Exercise 2


Step 1: Generating RISC-V Assembly File
Executed "riscv32-unknown-elf-gcc -S GCD.c -o GCD.S" command.
Output: GCD.S assembly file generated.

Step 2: Generating RISC-V Machine Code
Executed "riscv32-unknown-elf-as GCD.S -o GCD.o" command.
Output: GCD.o object file generated.

Step 3: Linking Object File to Generate Executable
**Executed** "riscv32-unknown-elf-ld GCD.o -o GCD_exe" command.

**Output:**
riscv32-unknown-elf-ld: warning: cannot find entry symbol _start; defaulting to 00010094
riscv32-unknown-elf-ld: GCD.o: in function `main':
GCD.c:(.text+0xa0): undefined reference to `printf'

**Executed** "riscv32-unknown-elf-ld GCD.o -o GCD_exe /opt/riscv/riscv32-unknown-elf/lib/crt0.o \
/opt/riscv/lib/gcc/riscv32-unknown-elf/14.2.0/crtbegin.o --start-group -lc -lgloss --end-group \
-L/opt/riscv/lib/gcc/riscv32-unknown-elf/14.2.0 -lgcc /opt/riscv/lib/gcc/riscv32-unknown-elf/14.2.0/crtend.o" command

**Output:** 
GCD_exe executable generated.

Step 4: Running the Executable on Spike with Proxy Kernel
Executed "spike --isa=rv32i pk GCD_exe" command.
Output: GCD of 64 and 16 is 16

Step 5: Running the Program in Debug Mode on Spike
Executed "spike -d --isa=rv32i pk GCD_exe" command.
Output: Debug mode activated, allowing instruction-by-instruction execution.
